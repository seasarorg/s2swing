はじめに

このページでは、チュートリアルを始めるにあたっての留意事項を記述します。

Swing Application FrameworkおよびBeans Bindingとの関係

S2Swingは、Swing Application FrameworkおよびBeans Bindingのラッパーとして位置付けられています。 したがって、本チュートリアルの内容は、Swing Application FrameworkとBeans Binding自身の機能説明を多く含むことになります。 機能がどのライブラリに属するかを常に示すと説明が煩雑になるため、本チュートリアルでは、一律「S2Swingの使用方法」として記述を行うこととします。

NetBeansについて

NetBeansでは、Swing Application FrameworkおよびBeans Bindingをベースとしたアプリケーション開発モードが用意されています。 この開発モードでは、本チュートリアルで明示的に用意するpropertiesファイルが自動的に作成されるなど、特有の挙動が見られます。 本チュートリアルは、IDEによる特段のサポートがない環境を前提として記述されていますので、NetBeansをお使いの方は、実際のIDEの動作に応じて、適宜読み替えをお願いします。
